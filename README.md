# PracticaMetodos

En este proyecto se hacen varias practicas de metodos para ver la forma de trabajar con ellos y su flujo de trabajo.
Sirve más como apunte y para saber que los conceptos aprendidos quedan bien aplicados.

## Mensaje
Es el primer metodo, solo imprime un mensaje en pantalla, las veces que este sea instanciado

## SumaNumeros
Genera la suma de dos numeros, como se puede ver este programa no devuelve nada por lo que la suma queda vacia 

## SumaTresNumeros
Genera la suma de tres numeros pero esta vez si se retorna un valor, lo que significa que puede ser utilizado más tarde

## Divide 2 numeros
Este metodo devuelve la division aqui ha que aclarar algo y es que si las variables fueran int no funcionaria ya que para poder procesar se requeieren 
que sean de tipo flotante, por lo que se cambia a tipo double

# Sobrecarga de Metodos
Para hacer una sobrecarga de metodos, o sea tener metodos que tengan el mismo nombre se tiene que 
hacer una sobrecarga de metodos esto es que los metodos que se llaman igual tengan diferentes tipos
o que tengan diferente numero de variables en este caso se hace esto con el metodo Multiplicacion el 
cual cuanta con 2 sobrecargas diferentes.

## Multiplicacion
Los metodos multiplicacion se hacen usando la funcion fleca para que sean mas legibles.
Estos son 2 enteros, 2 doubles o 3 enteros para que se entienda la idea
